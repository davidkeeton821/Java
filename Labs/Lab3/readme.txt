Loading and saving lists uses .txt format. To input tasks to a list through .txt format
use comma separated format.

Example:
Payroll,4,3 31 2018,y

Field 1) Name
Field 2) Priority, 1-4, higher number is higher priority
Field 3) Date, Month Day Year, these inner fields are separated by spaces
Field 4) Completed status, y- completed, n- not completed
